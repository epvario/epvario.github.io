Software

  • Firmware written in RUST, providing great memory safety, fast code execution and a broad compatibility and easy to ugprade
    • just drag and drop a firmware file to the USB connected vario from your PC or mobile
  • Numerical Variometer on the display
  • Altitude aMSL
  • Altitude above Ground (using SRTM elevation data)
  • Groundspeed and Glideratio
  • Compass with flight direction
  • Windspeed- and direction estimator
  • FAI CIVL approved IGC tracklog signing (G-record)
  • Flight and Landing detection, Flighttimer and Post-Flight Statistics

Developer quickstart:

  1. Install the Rust toolchain using rustup: https://rustup.rs/.
  2. Install the nightly toolchain and add the target to build for the RP2040:
rustup toolchain install nightly
rustup target add thumbv6m-none-eabi
  1. To build the firmware to flash it via the BOOTSEL mode, you need elf2uf2-rs:
cargo install elf2uf2-rs
  1. To build the firmware to flash it with the BOOTSEL mode, from the firmware folder run:
cargo build --release
elf2uf2-rs ./target/thumbv6m-none-eabi/release/ep-vario-firmware ep-vario-firmware.uf2

With the EPVario turned off, connect it to your computer using a USB-C cable. Hold the Up button and press and hold the power button for 2 seconds. A mass storage device should show up on your computer. Drag and drop the ep-vario-firmware.uf2 file that was just created to the device. It will automatically dismount the device and install the firmware.
